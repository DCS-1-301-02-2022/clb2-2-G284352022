- ## 情報工学科とは？
この学科では主にコンピュータについて学ぶことができます。
ハードウェアなどのコンピュータの構造から、
プログラミングなどのコンピュータの活用まで幅広く学ぶことができます。
また、**初心者でも経験者でも、それぞれにあった授業を選択できます。**
